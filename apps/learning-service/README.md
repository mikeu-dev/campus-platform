# Learning Service

This microservice powers the Learning Management System (LMS) features of the Campus Platform.

## Responsibilities
*   **Class Streams**: Feeds and announcements for classes.
*   **Assignments**: Creating, submitting, and grading assignments.
*   **Real-time Chat**: Messaging between students and lecturers.
*   **Materials**: Distribution of learning resources.

## Development

Run the service locally:
```bash
npm run dev -w learning-service
```
