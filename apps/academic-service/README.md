# Academic Service

This microservice handles core academic logic for the Campus Platform.

## Responsibilities
*   **Course Management**: Creation and maintenance of course catalog.
*   **Enrollment**: Handling student course selection (KRS/IRS).
*   **Grading**: Managing student grades and transcripts (KHS).
*   **Scheduling**: Managing academic calendars and class schedules.

## Development

Run the service locally:
```bash
npm run dev -w academic-service
```
